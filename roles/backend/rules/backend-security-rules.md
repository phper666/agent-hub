## Backend-Specific Security Rules

> 全局安全规则见 `.shared/rules/security-rules.md`
> 以下是后端开发额外需要遵守的安全规范

1. ORM 查询必须使用参数绑定，禁止 raw SQL 拼接
2. 所有 Service 层方法必须在入口处校验参数类型和范围
3. 敏感操作（删除、支付、权限变更）必须记录审计日志
4. 数据库连接必须使用连接池，禁止手动创建连接
5. API 返回数据必须经过序列化过滤，不暴露内部字段
6. 缓存 key 必须有命名空间前缀，防止冲突
7. 分布式锁必须设置超时时间
8. 消息队列消费必须做幂等处理
9. 定时任务必须防止重复执行
10. 配置变更必须支持热更新，不需要重启服务
