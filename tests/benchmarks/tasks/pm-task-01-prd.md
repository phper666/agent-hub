# PM Task 01: PRD — 电商购物车功能

## 角色
PM (产品经理)

## 输入 Prompt

```
你是一个产品经理。请为以下需求编写一份 PRD：

## 需求背景
一个电商平台需要新增购物车功能，让用户在浏览商品时可以：
- 将商品加入购物车
- 修改购物车中商品的数量
- 删除购物车中的商品
- 查看购物车总价

## 要求
- 按照标准 PRD 格式编写（问题描述、目标用户、核心场景、功能需求、验收标准、边界条件）
- 每个功能需求需包含 Given/When/Then 格式的验收标准
- 考虑以下场景：未登录用户、商品库存不足、优惠券叠加、多设备同步
- 输出到 docs/current/requirements/PRD.md
```

## 评估重点
- PRD 结构完整性（是否包含所有标准章节）
- 边界条件覆盖率（是否考虑了库存不足、登录状态、优惠券等）
- 验收标准的可执行性（Given/When/Then 是否具体可测）
- 用户故事质量（是否包含角色、目标、价值）
